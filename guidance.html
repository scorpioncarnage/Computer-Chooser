Guidance

Providing detailed instructions for what each component of a computer does and what that means to you

CPU/Processor

CPU/Processor

The Central Processing Unit (otherwise known as the CPU or Processor) is the main "brains" of the computer. This is an important component as it is mainly responsible for the speed and responsiveness, as well as the longevity of the computer.

Recommended midrange chips are:
Intel Core i5 & Core i7
AMD Ryzen 5 & Ryzen 7
Apple Silicon M1.

RAM

Random Access Memory

RAM is a vital part of a computer. It is mainly helpful in multitasking as it provides more space for the CPU to store data as it processes different tasks.

The standard amount for a normal user is 8 gigabytes, but 16 gigabytes would increase longevity. This component is the easiest to upgrade so keep in mind that most laptops allow for upgrading this part later on in the life of the computer.

Hard Drive

Hard Drive

Hard drives come in many forms and types. Simply put, the hard drive is the warehouse for storing files, applications, and all data the computer needs to store.

The best choice for a computer is an SSD (Solid State Drive) as it provides the best performance, but it is more expensive than HDDs (Hard Disk Drive) or HHDs (Hybrid Hard Drive).

Operating System

Operating System

The operating system is one of the most important factors for how users interact with the computer. Many users are used to Windows. But for new computer owners, Apple MacOS has a very small learning curve and Google ChromeOS is the most basic operating system.

Popular Operating Systems:
Windows (well-known)
MacOS (intuitive)
ChromeOS (extremely simple)

GPU/Graphics

Graphics Card

GPUs have a similar structure to CPUs but instead are dedicated to processing and outputting to the display, leaving more processing power to the CPU to work on processing other functions of the computer. GPUs are not mandatory for computers, but are mostly needed by gamers and users who work in media production.

Recommended midrange chips are:
Nvidia RTX 2060 Super & RTX 3060
AMD Radeon 5700 XT & 6600 XT

Display

Display

One of the most overlooked components of a computer is actually the main way that users interact with the device. There are 2 main aspects of a display to pay attention to: resolution and type.
The resolution defines how crisp and clear an image will look. A great choice for the average user is to find a display with 1080p display or above.
Most displays are either LCD or OLED. While OLED has better color accuracy, LCD is cheaper and more durable.
